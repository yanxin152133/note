# 1. 注释、标识符命名规则及Java中的关键字
## 1.1. Java注释
```
1. 单行注释：//
2. 多行注释：/*。。。。。*/
3. 文档注释：/**。。。*/
```
    
## 1.2. 标识符命名
```
标识符定义：Java语言中，对于变量、常量、函数、语句块也有名字，我们统统称为Java标识符。

标识符作用：标识符是用来给类、对象、方法、常量、接口和自定义数据类型命令的。
     
标识符命名规则：Java标识符由数字、字母和下划线（_），美元符号（$）组成。在Java中是区分大小写的，而且还要求首位不能是数字。最重要的是，Java关键字不能当作Java标识符。
```
